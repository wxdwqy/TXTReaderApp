# 极简TXT阅读器

一款纯粹的本地 TXT 文件阅读与编辑工具。不联网、不同步、不打扰。

## 为什么做这个？

手机上的记事本应用，要么功能太多太重，要么需要注册登录、云同步、推送广告。  
而有时候你只是想安安静静地看看小说、改改文字，仅此而已。

这个 APP 只做一件事：**读写 TXT 文件**。

## 功能

- 📂 **文件管理** — 扫描指定文件夹，列出所有 TXT 文件
- 📖 **阅读模式** — 干净的纯文本显示，支持字体大小、行间距调节
- ✏️ **编辑模式** — 直接修改并覆盖保存，像电脑上的记事本
- 🎨 **主题切换** — 白天 / 夜间 / 护眼 / 绿色 四种主题
- 📁 **双文件夹** — 支持同时管理两个文件夹（可自定义）
- 📄 **新建/删除** — 在 APP 内直接创建和删除文件

## 特点

- **纯本地** — 不需要网络权限，数据不离开你的手机
- **体积小** — APK 不到 3MB
- **零权限** — 只申请存储权限，用于读写文件
- **无广告** — 没有，以后也不会有
- **覆盖保存** — 编辑后直接保存回原文件，不会产生一堆副本

## 文件夹说明

APP 默认扫描两个文件夹（可在代码中修改）：

| 文件夹 | 路径 | 用途 |
|--------|------|------|
| 主文件夹 | `Documents/TXTReader/` | 个人文件 |
| 小说文件夹 | `Download/BaiduNetdisk/_pcs_.workspace/QCLAW/小说/` | 百度网盘同步的小说 |

首次打开 APP 会自动创建这些文件夹。

## 技术实现

- **Android WebView** + **JavascriptInterface** 桥接
- HTML/CSS/JS 负责界面，Java 负责文件读写
- 通过 `@JavascriptInterface` 注解让 JavaScript 调用原生文件操作
- 支持 Android 7.0+ (API 24+)

### 核心架构

```
┌─────────────────────────────┐
│         HTML/JS 界面         │  ← 负责显示和交互
├─────────────────────────────┤
│    FileBridge (Java)        │  ← JavascriptInterface 桥接
├─────────────────────────────┤
│    Android 文件系统          │  ← 实际读写 TXT 文件
└─────────────────────────────┘
```

## 构建

项目使用 GitHub Actions 自动构建。推送到 `main` 分支会自动生成 APK。

本地构建：
```bash
gradle assembleDebug    # Debug 版
gradle assembleRelease  # Release 版
```

## 安装

从 [Releases](https://github.com/wxdwqy/TXTReaderApp/releases) 下载 APK，传到手机安装即可。

> 注意：需要在系统设置中允许"安装未知来源应用"。

## 自定义

如果想修改默认文件夹，编辑 `FileBridge.java` 中的 `getMainDir()` 和 `getNovelDir()` 方法。
